🌐 Personal Portfolio Website

A modern and responsive personal portfolio website showcasing my projects, skills, and achievements as an aspiring software developer.

🔗 Live Demo: https://kavibharathi-98-portfolio.netlify.app/


 🚀 Features

* Clean and modern UI with smooth animations
* Responsive design (works on mobile, tablet, desktop)
* Projects section with real-world applications
* Skills and tools showcase
* Certifications included
* Easy navigation and minimal design

 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript
